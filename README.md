# DailyDict Vocab

Chrome Extension giúp lưu và ôn tập từ vựng từ trang [dailydictation.com](https://dailydictation.com/).

## 🚀 Tính năng
- **Bôi đen để tra từ:** Tự động hiển thị nghĩa English-Vietnamese và ví dụ.
- **Lưu từ vựng:** Lưu từ kèm theo ngữ cảnh (bài học đang nghe).
- **Ôn tập (SRS):** Thuật toán Spaced Repetition giúp ghi nhớ từ vựng hiệu quả qua Flashcards.
- **Dashboard:** Theo dõi tiến độ học tập, streak và biểu đồ 7 ngày.

## 🛠 Cài đặt (Developer Mode)
1. Tải thư mục này về máy.
2. Mở trình duyệt Chrome, truy cập `chrome://extensions/`.
3. Bật **Developer mode** (Góc trên bên phải).
4. Nhấn **Load unpacked** và chọn thư mục `extension/` trong dự án này.

## 📁 Cấu trúc thư mục
- `extension/`: Mã nguồn của Chrome Extension.
- `webapp/`: Giao diện Dashboard và Flashcards (mở trong tab mới).
